# Apie projektą

„City Game“ yra lengvas eksperimentinis miesto kūrimo žaidimas naršyklei.

Projektas kuriamas kaip smagus sandbox tipo žaidimas, kuriame galima laisvai tyrinėti, statyti ir eksperimentuoti su miesto aplinka.

Jokių sudėtingų taisyklių — tiesiog miesto kūrimas ir geras laiko praleidimas.

# Paleidimas

Projektas yra statinis, todėl pakanka atsidaryti:

```bash
index.html
```

arba paleisti paprastą lokalų serverį:

```bash
python -m http.server
```

Tada naršyklėje:

```bash
http://localhost:8000
```
